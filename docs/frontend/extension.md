# Extension

Extension navigateur = capture rapide offre.

## Role
- Lit page offre.
- Envoie titre, company, lien, notes vers backend.
- Evite saisie manuelle.

## Cote user
- Ouvre popup ou options.
- Configure api et compte.
- Lance creation offre depuis page courante.

## Effet cote backend
- Cree ou met a jour offre.
- Lance suite du flux candidature.
